# <img src="https://flagcdn.com/w40/us.png" width="40"> 🧩 ST-Cookies Plugin for GLPI

- ST-Cookies adds a cookie preferences banner to the GLPI login flow and provides an administrative page to configure the consent message displayed to users.

## Features

- Cookie consent banner on the login screen
- Separate handling for required and optional cookies
- Optional consent for **Remember me**
- Administrative page to customize the banner message
- Log records with date/time, IP, user (when available), and accepted categories

### 📋 Compatibility

- 📥 Plugin for GLPI version 10 (Coming soon)
- 📥 [Plugin for GLPI version 11](https://github.com/serviceticst/stcookies/releases/tag/v2.0.0)

## Installation

1. Copy the **stcookies** folder from the plugin package to the GLPI plugins directory.
2. Clear the GLPI cache.
3. Enable the plugin in **Setup > Plugins**.
4. Open the plugin configuration page and adjust the consent message if needed.

## Logs

- By default, the plugin records consent events in a log file, according to your environment configuration.
